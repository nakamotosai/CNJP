# cn.saaaai.com —— 从日本看中国

### 🇯🇵 实时 AI 日本涉华新闻聚合平台

这是一个全自动化的新闻聚合平台，旨在打破语言、国界与信息茧房，让用户能够实时、客观、直观地了解日本主流媒体眼中的中国。

---

## 🚩 核心特色 (Core Features)

*   **🤖 AI 智能编辑部**
    *   **每日简报**：首页顶端展示 AI 自动生成的当日「中日关系态势定调」、「关键事件」及「风向预测」。
    *   **深度解读**：每篇文章均支持一键生成 AI 深度分析，涵盖事实拆解、客观背景、多维评价及核心总结。
*   **📡 全网自动化扫描**
    *   每小时轮询抓取日本全网主流媒体，新闻记录永久保存在云端，支持海量历史归档查询。
*   **🎥 实时现场 (Live)**
    *   内置以东京为主的日本各大人气景点、交通枢纽的实时直播信号，直观感受邻国的人气脉动。
*   **🌍 灾害脉动 (Disaster)**
    *   集成交互式实时地震地图，结合全局提醒机制（M6以上地震），第一时间掌握日本自然灾害动态。
*   **📱 极极致 PWA 体验**
    *   基于渐进式 Web 应用架构，支持手机端「添加到主屏幕」以及 PC 端作为独立应用安装，获得原生 App 级流畅体验。

---

## 🛠️ 技术实现 (Tech Stack)

*   **前端**：Next.js 14 (App Router), Tailwind CSS, Framer Motion
*   **存储**：Cloudflare R2 (S3 兼容存储)，实现无数据库的极速访问
*   **后端**：Python + GitHub Actions (自动化流水线), FastAPI (AI 接口)
*   **AI 引擎**：Gemini / Claude / Ollama (私有化部署)，实现高精度的摘要与分析
*   **特别说明**：本项目主要由站长通过 **Gemini 2.0 / Claude 3.5 opus** 等 AI 工具全程无代码（No-code AI prompting）辅助完成编写。

---

## 💡 读者使用说明

1.  **国内访问**：针对大陆网络环境优化了 CDN 路由，无需魔法即可访问。
2.  **内容提示**：新闻原文链接和部分视频直播源直接调用自日本第三方平台，在某些网络状态下可能无法直接打开。
3.  **翻译说明**：本站自动翻译全部新闻标题并支持简繁切换。AI 翻译错误在所难免，重大事件建议查阅日文原文。
4.  **添加到主屏幕**：建议移动端用户将网页添加到桌面，以获得完整的沉浸式阅读体验。
